# 测试字体文件

将测试用的 TTF 字体文件放在此目录。

## 快速测试字体

项目根目录包含小型测试字体（约 0.6 MB），适合快速测试：

- `NotoSans-Light.ttf` - Noto Sans 轻量版
- `NotoSans-Regular.ttf` - Noto Sans 常规版

这些字体文件小，处理速度快，适合开发和调试时使用。

## CJK 字体（用于完整测试）

从 [Google Fonts](https://fonts.google.com/) 下载：

- Noto Sans JP (日文)
- Noto Sans KR (韩文)
- Noto Sans SC (简体中文)

这些字体有大量重叠的 CJK 字符，非常适合测试字形检测和去重功能。
